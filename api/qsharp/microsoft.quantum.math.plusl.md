包： [Microsoft 量子. 标准版](https://nuget.org/packages/Microsoft.Quantum.Standard)


返回两个输入之和。

```qsharp
function PlusL (a : BigInt, b : BigInt) : BigInt
```


## <a name="input"></a>输入

## <a name="output--bigint"></a>输出： [BigInt](xref:microsoft.quantum.lang-ref.bigint)
