包： [Microsoft 量子. 标准版](https://nuget.org/packages/Microsoft.Quantum.Standard)


计算数字的以2为底的对数。

```qsharp
function Lg (input : Double) : Double
```


## <a name="input"></a>输入

### <a name="input--double"></a>输入： [Double](xref:microsoft.quantum.lang-ref.double)
