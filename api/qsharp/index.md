# <a name="q-libraries"></a>Q# 库

Q# 标准库包括用于开发量子程序的操作、函数和用户定义的类型。 还包括了在标准库之上构建的另外三个库- 化学库、数字库和机器学习库。

有关 Q# 库的详细信息，请参阅 <xref:microsoft.quantum.libraries>。
