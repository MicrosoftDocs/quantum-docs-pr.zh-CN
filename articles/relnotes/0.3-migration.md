<span data-ttu-id="af635-141">最后一个 set 语句使用新的三元运算符，该运算符随0.3 更新引入，以支持条件表达式。</span><span class="sxs-lookup"><span data-stu-id="af635-141">The last set statement uses a new ternary operator that is introduced with the 0.3 update to support conditional expressions.</span></span>

<span data-ttu-id="af635-144">和中的分配 `using` `borrowing` 新适用于单个 qubits、qubit 数组和其所在的嵌套元组：</span><span class="sxs-lookup"><span data-stu-id="af635-144">Allocations within `using` and `borrowing` are newly possible for single qubits, qubit arrays, and nested tuples thereof:</span></span>
